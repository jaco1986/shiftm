New AWS EC2 Medata Transform

Enrich your events with EC2 metadata
type: new featuredomain: transformstransform: aws_ec2_metadata 1325

Are your events the laughing-stock of the data warehouse? Then enrich them with our brand spanking new aws_ec2_metadata transform.

Configuration isn't complicated, just add and hook up the transform. If you don't want all enrichments added then whitelist them with the fields option:

[transforms.fill_me_up]
  type = "aws_ec2_metadata"
  inputs = ["my-source-id"]
  fields = [
    "instance-id",
    "local-hostname",
    "public-hostname",
    "public-ipv4",
    "ami-id",
    "availability-zone",
    "region",
  ]

For more guidance get on the reference page.
