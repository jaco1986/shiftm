Vector v0.6.0 Release Notes

The Long Stretch
Dec 12th, 2019 / 6 months ago

The Vector team is pleased to release 0.6.0. This is a initial dev release. Checkout the Highlights, Changelog, and What's Next section for more detail.

#Highlights

Noteworthy changes in this release

  • # Breaking Changes

    • Require `encoding` option for console and file sinks

      The `encoding` option is now required for these sinks

    • The `datadog` sink has been renamed to `datadog_metrics`

      This ensures that naming is consistent for the upcoming `datadog_logs` sink

  • # New Features

    • ARMv7 & ARM64 Support on Linux

      These architectures are widely used in embeded devices & servers

    • Windows Support Is Here!

      Bringing the performance and reliability of Vector to Windows

    • Unit Testing Your Vector Config Files

      Treating your Vector configuration files as code

    • Use Custom DNS Servers

      Point Vector to custom DNS servers

    • New Kubernetes Source (alpha)

      Early adopters have a new Kuberenetes source to chew on

    • New AWS EC2 Medata Transform

      Enrich your events with EC2 metadata
