Vector v0.8.0 Release Notes

A Wholesome Platter Of Features
Feb 25th, 2020 / 4 months ago

  • This is an outdated version. Outdated versions maybe contain bugs. It is recommended to use the latest version (0.9.2).

The Vector team is pleased to release 0.8.0. This is a initial dev release. Checkout the Highlights, Changelog, and What's Next section for more detail.

#Highlights

Noteworthy changes in this release

  • # Enhancements

    • Automatically merge partial Docker events

      Docker splits long messages by default, and now Vector merges them back for you

    • Improved Multiline Support In The File Source

      Merge multiple lines together based on rules

  • # New Features

    • Support For Multiple Config Files

      A better way to manage complex Vector configurations

    • Introducing Vector's Global Log Schema

      Set defaults for Vector's common log key names

    • Lots of New Sinks!

      7 new sinks have been added to Vector

    • New Rename Fields Transform

      An easier way to rename fields

    • New Swimlanes Transform

      Split log streams with ease
